* Disable = 0, Enable = 1
* 'SQL Enabled' shows if the programs are Recording to a SQLite3 Databases

* 'Interval Time' is the Duration between recording readings to the SQLite3 Database
** Measured in Seconds. Default is 300 Seconds (5 min)

* 'Trigger Time' is the Duration between 2 Sensor readings, which are checked for a difference more then the Variance
** Measured in Seconds. Default is 0.15 Seconds (150ms)

* 'Custom' shows if you have sets your own Variances
* 'Acc' shows the currently set Accelerometer Variance
* 'Mag' shows the currently set Magnetometer Variance
* 'Gyro' shows the currently set Gyroscopic Variance

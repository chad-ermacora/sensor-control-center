* Disable = 0, Enable = 1

* 'Interval Time' is the Duration between recording, into the Interval SQL Database
* ** Measured in Seconds. Default is 300 Seconds (5 min)
* 'Trigger Time' is the Duration between readings, which are checked against the Variance
* ** Measured in Seconds. Default is 0.15 Seconds (150ms)

* 'SQL' shows if the Sensor is Recording to SQLite3 Databases
* 'Custom' shows if the sensor is using manually set Variances

* Default Trigger Variances are based on the Sensors in use

* 'Acc' shows the currently set Accelerometer Variance
* 'Mag' shows the currently set Magnetometer Variance
* 'Gyro' shows the currently set Gyroscopic Variance

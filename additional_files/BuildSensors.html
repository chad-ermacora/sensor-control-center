Building a Sensor Unit to work with KootNet Sensors

** WIP: This will be updated with better instructions as time goes on **


Currently Supported Sensor’s


Raspberry Pi products: Sense HAT


Pimoroni products: BH1745, BME680, Enviro pHAT, LSM303D, VL53L1X


Note: Additional Sensors can be added fairly easily, so long as they support the Raspberry Pi and Python3



Setting up a new Sensor


Step 1. Purchase a Raspberry Pi (Get either a 3B+ or Zero W, depending on your needs)


Step 2. Purchase desired Sensors from the supported list above (Or buy and program in your own)


Step 3. Install your Sensor(s) on to the Raspberry Pi


Step 4. Install Raspbian on your Raspberry Pi


Step 5. Make sure you are connected to the Internet and run the following command in a terminal to download and start the installer

Please note, this will automatically reboot the Raspberry Pi after completion


wget http://dragonwarz.net/Stuffz/SensorUpgrades/upgrade/install_sensors.sh && sudo chmod +x install_sensors.sh && sudo bash install_sensors.sh && sudo reboot



After following the onscreen Instructions, your Sensor Unit should be ready to go!
